# 基于大模型的个性化资源生成与学习多智能体系统

## 项目简介

本项目为第十五届中国软件杯大赛 A3 赛题作品，旨在构建一个基于大模型的个性化资源生成与学习多智能体系统，实现高等教育领域的"因材施教"数字化落地。

## 核心功能

1. **对话式学习画像自主构建**：通过自然语言对话自动抽取学生特征，构建动态学生画像
2. **多智能体协同的资源生成**：不同角色智能体协作生成多模态学习资源
3. **个性化学习路径规划和资源推送**：基于画像的动态学习路径规划与精准推送
4. **智能辅导**：即时多模态答疑解惑
5. **学习效果评估**：多维度学习效果评估与动态优化

## 技术栈

- **前端**：Vue 3 + Element Plus
- **后端**：Python + FastAPI
- **多智能体框架**：AutoGen / CrewAI
- **大模型**：讯飞星火大模型 API
- **数据库**：SQLite / PostgreSQL
- **向量数据库**：ChromaDB

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+

### 后端启动

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```
