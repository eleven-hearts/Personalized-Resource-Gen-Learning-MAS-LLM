# 测试说明书

## 1. 测试概述

### 1.1 测试目的

验证系统各功能模块的正确性、稳定性和性能表现，确保系统满足赛题要求。

### 1.2 测试环境

- **操作系统**：Windows 11 / Ubuntu 22.04
- **Python**：3.10+
- **Node.js**：18+
- **浏览器**：Chrome 120+, Edge 120+

## 2. 功能测试

### 2.1 用户认证模块

| 测试项 | 测试步骤 | 预期结果 |
|--------|----------|----------|
| 用户注册 | 输入用户名、邮箱、密码、专业，点击注册 | 注册成功，跳转登录页 |
| 用户登录 | 输入正确的用户名和密码 | 登录成功，跳转首页 |
| 密码错误 | 输入错误的密码 | 提示"用户名或密码错误" |
| 重复注册 | 使用已注册的用户名再次注册 | 提示"用户名已存在" |

| 测试项 | 测试步骤 | 预期结果 |

| 画像更新 | 继续对话，补充更多信息 | 画像动态更新 |
| 画像查看 | 进入"学习画像"页面 | 正确展示各维度信息 |

## 3. 性能测试

### 3.1 响应时间

| 测试项 | 目标响应时间 | 测试方法 |
|--------|-------------|----------|
| 对话响应 | < 3s | 发送消息，计时到收到回复 |
| 资源生成 | < 30s | 提交生成请求，计时到完成 |
| 页面加载 | < 2s | 打开各页面，计时到渲染完成 |

### 3.2 并发测试

| 测试项 | 并发数 | 目标 |
|--------|--------|------|
| 用户登录 | 50 | 成功率 > 99% |
| 对话请求 | 20 | 响应时间 < 5s |
