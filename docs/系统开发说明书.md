# 系统开发说明书

## 1. 项目概述

### 1.1 项目背景

本项目为第十五届中国软件杯大赛 A3 赛题作品，旨在构建一个基于大模型的个性化资源生成与学习多智能体系统。

### 1.2 技术栈

- **前端**：Vue 3 + Element Plus + Vite
- **后端**：Python + FastAPI + SQLAlchemy
- **多智能体**：自研智能体协调框架
- **大模型**：讯飞星火大模型 API
- **数据库**：SQLite（开发）/ PostgreSQL（生产）
- **向量数据库**：ChromaDB

### 2.2 多智能体架构

系统采用多智能体协同架构，各智能体职责如下：

| 智能体 | 职责 | 核心功能 |
|--------|------|----------|
| ProfileAgent | 学习画像构建 | 对话分析、画像抽取、动态更新 |
| ResourceAgent | 资源生成 | 文档/思维导图/题库/阅读/代码生成 |
| PathAgent | 路径规划 | 学习路径生成、阶段划分、资源推荐 |
| TutorAgent | 智能辅导 | 问题解答、多模态答疑、引导思考 |
| EvaluationAgent | 效果评估 | 学习分析、多维评估、策略优化 |

## 3. 数据库设计

### 3.1 用户表 (users)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| username | String | 用户名 |
| email | String | 邮箱 |
| hashed_password | String | 密码哈希 |
| full_name | String | 姓名 |
| major | String | 专业 |
| grade | String | 年级 |
| profile | JSON | 学习画像（6+维度） |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

### 3.2 资源表 (resources)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键 |
| user_id | Integer | 用户ID |
| title | String | 资源标题 |
| resource_type | String | 资源类型 |
| content | Text | 资源内容 |
| metadata | JSON | 元数据 |
| status | String | 状态 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

## 4. API 接口设计

### 4.1 认证接口

- `POST /api/v1/auth/register` - 用户注册
- `POST /api/v1/auth/login` - 用户登录

### 4.2 用户接口

- `GET /api/v1/users/me` - 获取当前用户
- `PUT /api/v1/users/me` - 更新用户信息
- `GET /api/v1/users/{id}/profile` - 获取学习画像
- `PUT /api/v1/users/{id}/profile` - 更新学习画像

### 4.3 资源接口

- `GET /api/v1/resources/` - 资源列表
- `POST /api/v1/resources/` - 创建资源
- `GET /api/v1/resources/{id}` - 资源详情
- `PUT /api/v1/resources/{id}` - 更新资源
- `DELETE /api/v1/resources/{id}` - 删除资源

### 4.4 智能体接口

- `POST /api/v1/agents/generate` - 生成资源
- `GET /api/v1/agents/status/{task_id}` - 查询任务状态
- `POST /api/v1/agents/learning-path` - 生成学习路径

### 4.5 对话接口

- `POST /api/v1/chat/message` - 发送消息
- `WS /api/v1/chat/ws` - WebSocket 对话

## 5. 开发环境搭建

### 5.1 后端

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
# 编辑 .env 配置讯飞API密钥
python main.py
```

### 5.2 前端

```bash
cd frontend
npm install
npm run dev
```

## 6. 部署说明

### 6.1 后端部署

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000
```

### 6.2 前端部署

```bash
cd frontend
npm run build
# 将 dist 目录部署到 Web 服务器
```
