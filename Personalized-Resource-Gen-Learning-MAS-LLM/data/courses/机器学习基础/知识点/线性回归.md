# 线性回归

## 基本概念

线性回归（Linear Regression）是一种用于建立输入变量（特征）与连续输出变量（目标）之间线性关系的监督学习算法。

## 数学模型

### 一元线性回归

假设输入属性只有一个，模型为：

$$f(x) = wx + b$$

其中 $w$ 为权重（斜率），$b$ 为偏置（截距）。

### 多元线性回归

当有多个输入特征时，模型为：

$$f(\mathbf{x}) = \mathbf{w}^T\mathbf{x} + b = w_1x_1 + w_2x_2 + ... + w_dx_d + b$$

## 损失函数

采用均方误差（Mean Squared Error, MSE）作为损失函数：

$$L(w, b) = \frac{1}{m}\sum_{i=1}^{m}(f(x_i) - y_i)^2 = \frac{1}{m}\sum_{i=1}^{m}(wx_i + b - y_i)^2$$

## 参数估计

### 最小二乘法

通过最小化损失函数来求解参数：

$$(w^*, b^*) = \arg\min_{(w,b)} \sum_{i=1}^{m}(f(x_i) - y_i)^2$$

对 $w$ 和 $b$ 求偏导并令其为0，可得闭式解：

$$w = \frac{\sum_{i=1}^{m}y_i(x_i - \bar{x})}{\sum_{i=1}^{m}x_i^2 - \frac{1}{m}(\sum_{i=1}^{m}x_i)^2}$$

$$b = \frac{1}{m}\sum_{i=1}^{m}(y_i - wx_i)$$

### 梯度下降法

当数据量较大或特征维度较高时，使用梯度下降迭代求解：

$$w := w - \alpha \frac{\partial L}{\partial w}$$

$$b := b - \alpha \frac{\partial L}{\partial b}$$

其中 $\alpha$ 为学习率。

## 正则化

### L1正则化（Lasso）

$$L_{lasso} = \frac{1}{m}\sum_{i=1}^{m}(f(x_i) - y_i)^2 + \lambda\sum_{j=1}^{d}|w_j|$$

### L2正则化（Ridge）

$$L_{ridge} = \frac{1}{m}\sum_{i=1}^{m}(f(x_i) - y_i)^2 + \lambda\sum_{j=1}^{d}w_j^2$$

## 优缺点

**优点**：
- 模型简单，易于理解和实现
- 计算效率高
- 结果具有较好的可解释性

**缺点**：
- 只能拟合线性关系
- 对异常值敏感
- 特征间存在多重共线性时效果不佳

## 应用场景

- 房价预测
- 销售预测
- 股票价格预测
- 气温预测
